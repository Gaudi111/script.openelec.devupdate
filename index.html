OpenELEC Dev Update

Unofficial update add-on for OpenELEC

Download this project as a .zip file Download this project as a tar.gz file

Changelog

OpenELEC has a built-in update mechanism for official releases but not for development builds. For those of us willing to test out new builds, report bugs, and follow through with the testing, this is an easy way to switch between builds, without leaving the XBMC interface. When a newer build is available at the specified source URL, a notification will be shown to enable you to install the update.

The add-on will download the (bzip compressed) tar file and extract the SYSTEM and KERNEL update images, and the associated md5 checksum files, into the /storage/.update folder.

If you have any comments or questions or suggestions for improvement, please post on the OpenELEC forum thread.

Installation

  1. Install my XBMC repository from xbmc.repo.leopold.zip.
  2. Install OpenELEC Dev Update from Leopold's Add-ons under Program Add-ons.
